# 智慧云脑诊疗平台 — 前端设计规范

基于医生门诊工作台 Demo 提炼，所有前端页面遵循此规范以保证视觉一致性。

---

## 1. 色彩系统

### 主色

| 变量 | 色值 | 用途 |
|---|---|---|
| `--bg-deep` | `#0b1929` | 深色背景（极少使用，仅全屏底色） |
| `--bg-sidebar` | `#0f2440` | 侧边导航背景 |
| `--bg-main` | `#f2f5f8` | 主内容区底色 |
| `--bg-card` | `#ffffff` | 卡片 / 面板背景 |
| `--bg-panel` | `#f8fafb` | 次级面板背景 |

### 文字

| 变量 | 色值 | 用途 |
|---|---|---|
| `--text-primary` | `#1a2b3c` | 标题、正文 |
| `--text-secondary` | `#5a6d80` | 辅助文字 |
| `--text-muted` | `#8899aa` | 占位、水印文字 |

### 功能色

| 变量 | 色值 | 用途 |
|---|---|---|
| `--accent` | `#1a7f8c` | 主按钮、选中态、强调 |
| `--accent-light` | `#e0f4f5` | 选中背景、浅色强调 |
| `--accent-dark` | `#0f5a65` | 按钮 hover、深色强调 |
| `--success` | `#2e9e5f` | 成功、完成状态 |
| `--warning` | `#e8933b` | 警告、过号状态 |
| `--danger` | `#d1464b` | 驳回、错误 |
| `--info` | `#3b7fd4` | 信息提示 |

### 边框

| 变量 | 色值 | 用途 |
|---|---|---|
| `--border` | `#e2e8ef` | 默认分割线、输入框边框 |

### 渐变

```
侧边 Logo：linear-gradient(135deg, #1a9fad, #0f7a88)
医生头像：linear-gradient(135deg, #1a7f8c, #39a0b0)

```

| 层级 | 字体 | 大小 | 粗细 | 用途 |
|---|---|---|---|---|
| 页面标题 | PingFang SC | 18px | 600 | 页面大标题 |
| 面板标题 | PingFang SC | 14-15px | 600 | 卡片/面板标题 |
| 卡片小标题 | PingFang SC | 12-13px | 600 | 区块标签（大写+letter-spacing） |
| 正文 | PingFang SC | 13-14px | 400 | 正文、病历内容 |
| 辅助文字 | PingFang SC | 11-12px | 400 | 标签、说明、时间戳 |
| 徽章 | PingFang SC | 12px | 600 | Badge |

## 3. 圆角

| 变量 | 值 | 用途 |
|---|---|---|
| `--radius-sm` | `6px` | 输入框、小按钮 |
| `--radius-md` | `10px` | 卡片、面板、列表项 |
| `--radius-lg` | `14px` | 大卡片、模态框 |
| 全圆 | `50%` | 头像、状态点 |
| Pill | `20-22px` | 圆角按钮、徽章 |

---

## 4. 阴影

| 变量 | 值 | 用途 |
|---|---|---|
| `--shadow-sm` | `0 1px 3px rgba(11,25,41,0.06)` | 卡片、面板默认 |
| `--shadow-md` | `0 4px 12px rgba(11,25,41,0.08)` | 悬浮卡片 |
| `--shadow-lg` | `0 8px 24px rgba(11,25,41,0.12)` | 模态框 |

阴影始终带蓝色底调 `rgba(11,25,41,x)`，不用纯黑。

## 5. 间距

| 层级 | 值 | 用途 |
|---|---|---|
| xs | `4-6px` | 紧凑元素间距 |
| sm | `8-10px` | 表单标签间距 |
| md | `12-16px` | 卡片内边距、按钮间距 |

## 6. 布局

### 三栏经典

```
┌──────┬─────────────┬──────────────┐
│ 64px │   320px     │  flex:1      │  360px
│ 导航  │  候诊队列    │  工作台       │  AI 助理
│ 固定  │  固定宽度    │  弹性填充      │  固定宽度
└──────┴─────────────┴──────────────┘
```

- 侧边导航：固定 `64px`
- 左面板：固定 `320px`
- 中央工作区：`flex: 1` 弹性填充
- 右面板：固定 `360px`

### 卡片网格

```
工作区内卡片 flex:1 等分，gap: 16px
内容区域 padding: 20px 24px
卡片内 grid-template-columns: 1fr 1fr（信息展示）
```

## 7. 组件规范

### 按钮

| 类型 | 样式 | 场景 |
|---|---|---|
| 主按钮 `.btn-primary` | 背景 `--accent`，白字，pill 圆角，`shadow` | 确认、保存、提交 |
| 次按钮 `.btn-ghost` | 白底 + `--border` 边框，灰字 | 取消、暂存 |
| 叫号按钮 `.call-btn` | 主色 + 放大 + 加阴影 | 唯一高优操作 |
| 侧边图标 `.sidebar-btn` | 透明底，hover 微亮 | 导航 |

### 状态点 `.status-dot`

```
⚫ waiting    #8899aa  静态
🟢 consulting #1a7f8c  脉冲动画 pulse 2s
🟠 passed     #e8933b  静态
```

脉冲动画 `@keyframes pulse`：`opacity 1 ↔ 0.4`，2s 循环。

### 徽章 `.badge`

```
底色 --accent-light + 文字 --accent-dark
font-size 12px, padding 3px 10px, border-radius 12px
```

### 卡片 `.card`

```css
background: var(--bg-card);
border-radius: var(--radius-md);
padding: 20px;
box-shadow: var(--shadow-sm);
```

### AI 建议卡片 `.ai-card`

```css
background: var(--bg-panel);
border-left: 3px solid var(--accent);  /* 默认青绿色 */
border-radius: var(--radius-md);
padding: 14px;
```

变体：
- `.ai-card.warning` → `border-left-color: --warning`，标题色 `--warning`
- `.ai-card.info` → `border-left-color: --info`，标题色 `--info`

### 模态框 `.modal`

```css
background: #fff;
border-radius: var(--radius-lg);
padding: 28px;
width: 480px;
box-shadow: var(--shadow-lg);
```
背景遮罩：`rgba(11,25,41,0.5)` + `backdrop-filter: blur(4px)`

### Toast

```css
position: fixed; top: 24px; right: 24px;
border-radius: var(--radius-md);
font-size: 13px; font-weight: 600; color: #fff;
```
- `.toast.success` → `--success`
- `.toast.info` → `--info`

### 滚动条

```css
width: 5px
thumb: #d0d8e0
track: transparent
```

## 8. 交互规范

| 场景 | 动效 | 时长 |
|---|---|---|
| hover | `transform: translateY(-1px)` + 阴影增强 | 0.2s ease |
| 选中态 | 边框变色 + 浅色背景 | 0.2s |
| 模态框 | 背景模糊 + fadeIn | 0.3s |
| Toast | translateY(-10px) → 0 + opacity 0→1 | 0.3s |
| 状态脉冲 | opacity 关键帧 | 1.5-2s 循环 |
| 输入框 focus | 边框变色 + `box-shadow` 光圈 | 0.2s |

**统一过渡函数：** `cubic-bezier(0.4, 0, 0.2, 1)`

## 10. 页面标题模板

每页顶部居中标题：

```html
<div class="page-header">
  <h1>页面名称</h1>
  <p class="subtitle">简要描述</p>
</div>
```

样式：
```css
.page-header { text-align: center; padding: 32px 0 16px; }
.page-header h1 { font-size: 20px; font-weight: 700; color: var(--text-primary); }
.page-header .subtitle { font-size: 13px; color: var(--text-muted); margin-top: 4px; }
```

## 11. 命名约定

| 前缀 | 用途 | 示例 |
|---|---|---|
| `.panel-` | 大区块面板 | `.panel-left` `.panel-center` `.panel-right` |
| `.card` | 卡片容器 | `.patient-card` `.emr-card` |
| `.btn-` | 按钮 | `.btn-primary` `.btn-ghost` |
| `.status-` | 状态指示 | `.status-waiting` `.status-active` |
| `.ai-` | AI 助理组件 | `.ai-card` `.ai-header` `.ai-input` |
| `.o-` | 医嘱组件 | `.o-icon` `.o-label` `.o-desc` |
