# 智慧云脑诊疗平台 - 核心流程活动图

```mermaid
stateDiagram-v2
    state "患者 - 诊前准备" as 患者
    state "医生 - 诊中枢纽" as 医生
    state "AI 助理医生" as AI助理
    state "医技/AI - 异步算力" as 医技
    state "医嘱路由" as 医嘱路由
    state "药房管理" as 药房
    state "管理员 - 后勤基座" as 管理员

    %% ===== 管理员线（橙色） =====
    state 管理员 {
        管理员登录 --> AI排班
        AI排班 --> 释放号源
        释放号源 --> 更新出诊计划
    }

    %% ===== 排班触发挂号 =====
    释放号源 --> 患者挂号汇入
    管理员线下挂号 --> 患者挂号汇入

    %% ===== 患者线（蓝色） =====
    state 患者 {
        患者注册登录 --> AI智能分诊
        AI智能分诊 --> 患者挂号汇入: 进入候诊池
        state 患者挂号汇入 <<join>>
        患者挂号汇入 --> 等待叫号
    }

    %% ===== 医生线（绿色） =====
    state 医生 {
        医生登录 --> 门诊工作台
        门诊工作台 --> 队列合并
        state 队列合并 <<join>>
        等待叫号 --> 队列合并
        管理员线下挂号 --> 队列合并
        队列合并 --> 叫号接诊
        叫号接诊 --> 问诊中
    }

    %% ===== AI 助理联动 =====
    state AI助理 {
        旁路启动 --> 监听音频流: 医生进入问诊时自动触发
        监听音频流 --> LLM意图识别
        LLM意图识别 --> 推送鉴别诊断
        推送鉴别诊断 --> 问诊中: 实时侧栏展示
        监听音频流 --> 推送用药提醒
        推送用药提醒 --> 问诊中
    }

    %% ===== 问诊分支 =====
    state 问诊中 {
        写电子病历
        开具检查单
        开具处方
    }

    问诊中 --> 开具检查单: 下达检查医嘱
    问诊中 --> 开具处方: 下达处方

    %% ===== 医嘱路由（同步确认 + 异步执行） =====
    state 医嘱路由 {
        持久化确认 --> 返回accepted: 同步 RPC
        返回accepted --> 问诊中: 医生看到确认
        持久化确认 --> 发布OrderIssued: 异步 Outbox
        发布OrderIssued --> 路由分发: Kafka 事件
    }

    开具检查单 --> 持久化确认
    开具处方 --> 持久化确认

    %% ===== 检查路径（红色） =====
    state 医技 {
        执行检查 --> 上传影像
        上传影像 --> AI影像识别
        AI影像识别 --> 生成AI报告
        生成AI报告 --> 报告回传
    }

    路由分发 --> 执行检查
    报告回传 --> 问诊中: 红点/弹窗提示

    %% ===== 处方路径 =====
    state 药房 {
        配药核验 --> 库存扣减
        库存扣减 --> 发药完成
    }

    路由分发 --> 配药核验
    发药完成 --> 问诊中: 处方状态更新

    %% ===== 结束就诊 =====
    问诊中 --> 结束就诊
    结束就诊 --> AI摘要归档: ConsultationAiSummaryEvent
    AI摘要归档 --> 病历归档

    %% ===== 异常路径 =====
    开具检查单 --> 驳回告警: OrderRejectedEvent
    开具处方 --> 驳回告警
    驳回告警 --> 问诊中

    叫号接诊 --> 患者过号: PASSED
    患者过号 --> 队列合并
```

## 泳道版（简化视角）

```mermaid
sequenceDiagram
    actor 患者
    actor 医生
    participant 门诊工作台
    participant AI助理
    participant 医嘱路由
    participant 医技
    participant 药房
    actor 管理员

    %% 排班 → 挂号
    管理员->>门诊工作台: 排班 + 释放号源
    患者->>门诊工作台: AI 分诊 → 挂号
    Note over 门诊工作台: 线上/线下队列合并

    %% 就诊
    医生->>门诊工作台: 叫号接诊
    门诊工作台->>AI助理: ConsultationStarted (Kafka)

    rect rgb(240, 255, 240)
        Note over 医生,AI助理: WebSocket 实时伴诊
        医生--)AI助理: 语音流上行
        AI助理--)医生: 鉴别诊断 + 用药建议推送
    end

    %% 开检查
    医生->>医嘱路由: 开具检查单 (同步 RPC)
    医嘱路由-->>医生: accepted ✓
    医嘱路由->>医技: OrderIssued (Kafka)
    医技->>医技: AI 影像识别
    医技->>门诊工作台: CtReportGenerated (Kafka)
    门诊工作台-->>医生: 报告弹窗提示

    %% 开药
    医生->>医嘱路由: 开具处方 (同步 RPC)
    医嘱路由-->>医生: accepted ✓
    医嘱路由->>药房: OrderIssued (Kafka)
    药房->>门诊工作台: OrderFulfilled (Kafka)

    %% 结束
    医生->>门诊工作台: 结束就诊
    门诊工作台->>AI助理: 生成摘要 (Kafka)
    AI助理->>门诊工作台: ConsultationAiSummary (Kafka)
```
